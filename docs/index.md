# 章节目录

## 第一篇：AI 的发展历程（第 1-3 章）

   - AI 寒冬的到来与反思
2. 机器学习的崛起（1990s-2010s）
   - 统计学习方法的发展
   - 深度学习技术的突破

   - Transformer 架构的革命性影响
   - GPT 系列模型的演进
   - AI 从专用到通用的转变
